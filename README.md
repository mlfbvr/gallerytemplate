# Gallery Template

## Template generation project for Bashgallery

This project provides a simple template for creating image galleries using Bashgallery. It includes a basic HTML structure and Tailwind CSS for styling.

## Features

- Responsive design using Tailwind CSS.
- Easy to customize HTML files.
- Simple build process using npm scripts.

### Prerequisites

- Node.js and npm installed on your machine.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/mlfbvr/gallerytemplate
   ```
2. Navigate to the project directory:
   ```bash
   cd gallerytemplate
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```

### Files

- `src/mainindex.html`: Main gallery HTML template.
- `src/albumindex.html`: Album view HTML template.
- `src/styles.css`: Tailwind CSS input file.
- `dist/`: Directory where the built files will be generated.

### Usage

1. Build the project:
   ```bash
   npm run build
   ```
2. The generated files will be available in the `dist/` directory.
3. Open `dist/mainindex.html` in your web browser to view the gallery template.

### Customization

- Modify the HTML files in the `src/` directory to customize the gallery layout.
